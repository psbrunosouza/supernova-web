# Button

O `Button` é um componente utilizado para permitir que o usuário execute ações na interface.
Ele é amplamente utilizado para confirmar ou cancelar ações, enviar formulários, abrir modais e navegar entre etapas de um fluxo.

## Uso

### Filled

Use `filled` para ações principais.

<Canvas of={ButtonStories.Filled} sourceState="shown" />

### Outline

Use `outline` para ações secundárias.

<Canvas of={ButtonStories.Outline} sourceState="shown" />

### Ghost

Use `ghost` para ações discretas.

<Canvas of={ButtonStories.Ghost} sourceState="shown" />

### Disabled

Botões desabilitados não permitem interação.

<Canvas of={ButtonStories.Disabled} sourceState="shown" />

### Primary

Use `Primary` como cor principal para exibição de conteúdo

<Canvas of={ButtonStories.Primary} sourceState="shown" />

### Accent

Use `Accent` como cor de destaque para ações secundárias e elementos que precisam chamar atenção sem competir com a ação principal.

<Canvas of={ButtonStories.Accent} sourceState="shown" />

### Success

Use Success para indicar uma ação que resulta em uma operação bem-sucedida ou em um estado positivo.

<Canvas of={ButtonStories.Success} sourceState="shown" />

### Warning

Use Warning para ações que exigem atenção ou podem ter consequências importantes.

<Canvas of={ButtonStories.Warning} sourceState="shown" />

### Error

Use Error para ações destrutivas ou irreversíveis.

<Canvas of={ButtonStories.Error} sourceState="shown" />
