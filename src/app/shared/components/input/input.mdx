# Input

O `Input` e um componente utilizado para capturar texto em formularios.
Ele exibe um label, suporta placeholder, estado desabilitado e conteudo projetado para prefixos ou icones de apoio.

## Uso

### Default

Use para capturar texto em campos simples do formulario.

<Canvas of={InputStories.Default} sourceState="shown" />

### With Prefix

Use conteudo projetado para adicionar prefixos ou icones sem alterar o valor digitado.

<Canvas of={InputStories.WithPrefix} sourceState="shown" />

### With Value

Use com `ngModel` quando o campo precisar refletir um valor controlado pelo formulario.

<Canvas of={InputStories.WithValue} sourceState="shown" />

### Disabled

Campos desabilitados permanecem visiveis, mas nao permitem interacao.

<Canvas of={InputStories.Disabled} sourceState="shown" />
